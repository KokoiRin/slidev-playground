# Slidev 初始化及运行命令记录

## 1. 安装 Slidev 及其主题依赖
```bash
npm install @slidev/cli @slidev/theme-seriph --save-dev
```
- **作用**：在当前项目（由 package.json 初始化）中安装 Slidev 命令行工具，并安装我们在 `slides.md` 中指定的 `seriph` 主题。这些依赖会保存在本地。

## 2. 启动 Slidev 开发服务器
```bash
npm run dev
```
- **作用**：运行我们在 `package.json` 的 `scripts` 中配置的 `dev` 脚本。这等同于执行 `slidev --port 3030`，它会启动一个开发服务器并在 3030 端口进行监听，提供热更新（HMR）功能。
